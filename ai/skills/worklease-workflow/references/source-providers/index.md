# Provider References

Load only the reference matching each caller-resolved provider kind. These references define provider deltas; [`contract.md`](../contract.md) remains normative.

- [`backlog-md.md`](backlog-md.md) — Backlog.md projects and tasks
- [`loose-markdown.md`](loose-markdown.md) — established Markdown work sources
- [`github-issues.md`](github-issues.md) — GitHub issue collections
- [`linear.md`](linear.md) — Linear teams, projects, and issues
- [`jira.md`](jira.md) — Jira sites, projects, and issues
- [`unknown.md`](unknown.md) — custom or unsupported provider kinds

A provider reference is not authorization. The caller supplies credentials, allowed operations, exact source scope, and any provider-specific status/dependency interpretation.
