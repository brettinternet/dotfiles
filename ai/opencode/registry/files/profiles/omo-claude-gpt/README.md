# Claude+GPT OMO profile

OpenCode profile for Anthropic Claude and OpenAI GPT Oh-My-OpenAgent routing.

- `ocx.jsonc` keeps the profile conservative.
- `opencode.jsonc` stays aligned with the top-level OpenCode setup.
- `oh-my-openagent.jsonc` routes OMO agents and categories across Claude Opus/Sonnet/Haiku and OpenAI GPT models.
